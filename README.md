## Description

NerdIT API

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start:dev

# test
$ ./test-run.sh
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## etc
```bash
# lint
$ npm run lint

# remove dist
$ npm run prebuild
```
